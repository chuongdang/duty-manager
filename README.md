### Overview

This project is a simple project to automate scheduling for staffs

#### Techniques
* PHP 7
* PhalconPHP (https://phalconphp.com)
* React
* Redux
* Material UI (https://material-ui.com)

### Local test

```bash
cd docker
./run.sh
```

### Frontend development and debug

```bash
cd public/dev
yarn start
```
